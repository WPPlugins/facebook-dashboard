Add a dashboard widget for interacting with Facebook.

== Description ==

This plug-in addes the Facebook for iGoogle gadget (published by Google) to your WordPress dashboard.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire `/facebook-dashboard` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why use an iGoogle gadget? =

Why reinvent the wheel?  The gadget already exists and is easy to embed in any page with an iframe - which is exactly how Google puts it in an iGoogle homepage.
